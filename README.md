# SystemOps Partner

React + Tailwind CSS で構築された、システムオペレーション パートナーのポートフォリオサイトです。

## 📋 概要

物流システム構築22年の経験と、ITベンチャーでのDXマネジメント経験を活かしたポートフォリオサイト。

### 主な特徴

- **React** + **Tailwind CSS** による最新のWebデザイン
- - **Lucide React** アイコンライブラリを使用した美しいUI
  - - レスポンシブデザイン対応
    - - 物流システム、DX推進、業務自動化の実績紹介
     
      - ## 🛠 技術スタック
     
      - - **フレームワーク**: React
        - - **スタイリング**: Tailwind CSS
          - - **アイコン**: Lucide React
            - - **その他**: JavaScript ES6+
             
              - ## 📁 プロジェクト構成
             
              - ```
                src/
                ├── components/
                │   └── Portfolio.jsx  # メインコンポーネント
                ├── App.jsx
                └── index.js
                ```

                ## 🚀 セットアップ方法

                ### 必要な環境
                - Node.js (v14以上)
                - - npm または yarn
                 
                  - ### インストール
                 
                  - ```bash
                    # リポジトリのクローン
                    git clone https://github.com/cocona000/system-ops-partner.git
                    cd system-ops-partner

                    # 依存関係のインストール
                    npm install

                    # 開発サーバーの起動
                    npm start
                    ```

                    ### ビルド

                    ```bash
                    npm run build
                    ```

                    ## 📄 ファイル説明

                    ### Portfolio.jsx
                    ポートフォリオサイトのメインコンポーネントです。以下のセクションを含みます：

                    - **Hero Section**: キャッチコピーと自己紹介
                    - - **Strengths Section**: 3つの強み（物流システム、DX、AI活用）
                      - - **Services Section**: 提供可能なサービス
                        - - **Works Section**: プロジェクト実績 (8件)
                          - - **Process Section**: 納品までの流れ
                            - - **FAQ Section**: よくあるご質問
                              - - **Contact Section**: お問い合わせ
                               
                                - ## 🎨 デザイン特徴
                               
                                - - グラデーション背景とホバーエフェクト
                                  - - アコーディオン形式のFAQ
                                    - - レスポンシブグリッドレイアウト
                                      - - スムーズなアニメーション
                                        - - 暗いテーマと明るいテーマの配色
                                         
                                          - ## 📞 お問い合わせ
                                         
                                          - 各セクションの「お問い合わせ」ボタンから、クラウドワークスやランサーズでの相談を受け付けています。
